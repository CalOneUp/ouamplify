Player Personas

Bartle's Taxonomy implementation with persona-specific UI variants

Persona Detection

Player personas are automatically detected based on user behavior patterns:

  • Achiever: High badge count, level, and impact score
  • Explorer: Original posts, customized content, exploration missions
  • Socializer: Peer badges received, comments generated, team participation
  • Killer: High leaderboard position, competitive ranking

Grid System

Uses Tailwind CSS grid utilities with responsive breakpoints:

  • Mobile: Single column (grid-cols-1)
  • Tablet: 2 columns (md:grid-cols-2)
  • Desktop: 12-column system (lg:grid-cols-12)
